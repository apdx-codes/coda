# Complete DApp Example

This example demonstrates a full-stack Solana DApp with Rust program and TypeScript/React frontend.

## Project Structure

```
complete-dapp/
├── program/           # Anchor Solana program
├── app/              # Next.js frontend
└── tests/            # Integration tests
```

## Features

- Counter program with increment/decrement
- React frontend with wallet connection
- Real-time state updates
- Complete TypeScript SDK
